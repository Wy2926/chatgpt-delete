# ChatGPT 批量管理器

Chrome 扩展（Manifest V3），用于批量删除 / 归档 ChatGPT 会话。

## 功能

- **批量删除 / 归档 / 取消归档**：一次操作任意数量的会话，操作前二次确认
- **友好的多选交互**：
  - 点击选择 / 取消选择
  - Shift + 点击区间选择
  - 鼠标拖动**框选**（按住 Alt 拖动为取消选择）
  - 全选 / 反选 / 清除选择 / 标题搜索过滤
- **限速与重试**：请求串行执行并限速；遇到 429 时按 `Retry-After` / 指数退避自动重试；失败项一键重试
- **进度反馈**：实时进度条与每项状态（处理中 / 完成 / 失败原因），支持暂停 / 继续 / 取消
- **不打扰**：UI 使用 Shadow DOM 隔离，只有一个右下角悬浮按钮，不影响 ChatGPT 页面正常使用
- **无需额外登录**：直接复用你在 chatgpt.com 的登录状态（同源请求），不收集、不上传任何数据

## 安装

1. 下载/克隆本仓库
2. 打开 Chrome，访问 `chrome://extensions`
3. 打开右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本仓库目录

## 使用

1. 打开并登录 [chatgpt.com](https://chatgpt.com)
2. 点击页面右下角的 🗂 悬浮按钮打开管理面板
3. 选择会话（点击 / Shift 区间 / 拖动框选 / 全选）
4. 点击「删除所选」或「归档所选」；「已归档」标签页中可取消归档或删除

## 原理

调用 ChatGPT 官方 `backend-api`（与网页端相同的接口）：

- `GET /backend-api/conversations` — 分页获取会话列表
- `PATCH /backend-api/conversation/{id}` — `{"is_visible": false}` 删除、`{"is_archived": true|false}` 归档/取消归档

访问令牌来自 `GET /api/auth/session`（同源 Cookie），仅在页面内存中缓存。

## 注意

- 删除操作不可恢复，请谨慎使用
- ChatGPT 接口有速率限制，大批量操作会自动放慢速度，请耐心等待
